# YOLOv8 Object Detection Project

## 📌 About
This project uses YOLOv8 for object detection.

## 📂 Files
- app.py – Main application file
- main.py – Core logic
- yolov8n.pt – Pretrained YOLOv8 model
- videos/ – Input videos

## ⚙️ Requirements
- Python 3.x
- Ultralytics YOLOv8
- OpenCV

Install dependencies:
pip install ultralytics opencv-python

## ▶️ Run the Project
python app.py
